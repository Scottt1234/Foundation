# US-accent narration

Drop the **US-accent** MP3s here, using the **same filenames** as
`assets/audio/` (e.g. `S01B00.mp3`, `S06F03.mp3`, `S10F00.mp3`). When a viewer
chooses **US** on the setup screen, the deck loads clips from this folder.

**Fallback:** any clip missing from this folder automatically falls back to the
base `assets/audio/` set, so you can populate the accent one file at a time
without ever leaving the deck silent.

See `assets/audio/README.md` for the full filename list and naming convention.
